# ChatGPT API Integrated Chatbot

## Açıklama

Bu proje, OpenAI ChatGPT API'sini kullanan basit ve etkili bir chatbot API'sidir. Express.js tabanlı backend, kullanıcıdan gelen mesajları OpenAI API'ye iletir ve dönen yanıtı kullanıcıya JSON formatında sunar.

## Proje Hedefleri

- OpenAI ChatGPT API entegrasyonu
- RESTful API tasarımı ve geliştirilmesi
- Hata yönetimi ve güvenli API anahtarı kullanımı
- Ölçeklenebilir ve modüler kod yapısı

## Kullanılan Teknolojiler / Kaynaklar

- Node.js
- Express.js
- Axios (HTTP istemcisi)
- dotenv (çevresel değişken yönetimi)
- OpenAI ChatGPT API
