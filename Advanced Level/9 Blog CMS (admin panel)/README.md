# Blog CMS (Admin Panel)

## Proje Hedefleri
- Kullanıcıların kayıt olup giriş yapabileceği, yetkilendirme sistemi olan güvenli bir API geliştirmek.
- Blog yazılarını oluşturma, düzenleme, silme işlemlerini rol bazlı erişim kontrolüyle yönetmek.
- JWT tabanlı oturum yönetimi ve şifrelerin bcrypt ile güvenli şekilde saklanması.
- Admin ve editor rollerinin farklı yetkilere sahip olması.
- Hata yönetimi ve güvenlik kontrolleri ile sağlam bir backend sunmak.

## Açıklama
Bu proje Node.js, Express ve MongoDB kullanılarak geliştirilmiş API tabanlı bir Blog CMS uygulamasıdır. Kullanıcılar kendilerini kaydedebilir, giriş yapabilir, rollerine göre blog yazılarını yönetebilirler. Yetkilendirme JWT ile sağlanmakta, şifreler bcrypt ile güvenli bir şekilde saklanmaktadır. Rol bazlı erişim ile admin ve editör kullanıcıları farklı yetkilere sahiptir.
