# Email Verification System

## Proje Hedefleri
- Kullanıcıların kayıt olurken e-posta adreslerini doğrulamalarını sağlamak.
- Güvenli ve token bazlı doğrulama sistemi oluşturmak.
- Kullanıcı deneyimini artırmak için doğrulama e-postası göndermek.

## Açıklama
Bu proje, Node.js ve Express kullanarak basit bir e-posta doğrulama sistemi sunar. Kullanıcılar kayıt olduklarında, kendilerine bir doğrulama bağlantısı içeren e-posta gönderilir. Kullanıcı bu bağlantıya tıklayarak hesaplarını doğrularlar. Sistem, token bazlı güvenlik ve süre sınırlaması uygular.
