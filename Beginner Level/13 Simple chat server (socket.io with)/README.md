## Proje Açıklaması
Bu proje, Socket.io kullanarak basit bir sohbet uygulaması geliştirmeyi amaçlar. Kullanıcılar birbiriyle gerçek zamanlı olarak mesajlaşabilir. Mesajlar sunucuya gönderildikçe, sunucu tüm bağlı istemcilere bu mesajı iletecektir. Kullanıcılar anonim olarak sohbet edebilirler.

## Ekip/Kaynaklar
- **Geliştirici**: [ Quenn Exe]
- **Teknolojiler**:
    - Node.js
    - Express.js
    - Socket.io

## Proje Hedefleri
- Gerçek zamanlı mesajlaşma.
- Kullanıcıların anonim olarak sohbet edebilmesi.
- Socket.io kullanarak sunucu ve istemci arasındaki verilerin iletilmesi.
