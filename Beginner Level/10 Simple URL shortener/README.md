## Proje Açıklaması
Bu proje, kullanıcıların uzun URL'leri kısaltmalarına olanak sağlayan basit bir URL kısaltıcı uygulamasıdır. Kullanıcılar, uzun URL'lerini girip kısaltılmış bir versiyonunu alırlar ve kısaltılmış URL'ye tıkladıklarında orijinal URL'ye yönlendirilirler.

## Ekip/Kaynaklar
- **Geliştirici**: [ Quenn Exe]
- **Teknolojiler**:
    - Node.js
    - Express.js
    - UUID (Benzersiz ID oluşturma)

## Proje Hedefleri
- Kullanıcıların URL'lerini kısaltma.
- Kısaltılmış URL'ye tıklandığında kullanıcıyı orijinal URL'ye yönlendirme.
