# API'den Veri Çekip Loglama

Bu proje, dış bir API'den veri çekip konsola loglama işlemini gerçekleştirir.

## Kullanılan Teknolojiler
- Node.js
- axios: HTTP istekleri yapmak için
- dotenv: Çevre değişkenlerini yönetmek için (isteğe bağlı)

## Başlangıç
