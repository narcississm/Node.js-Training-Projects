# ⚙️ .env Configuration System with dotenv

## Açıklama

Bu proje, `.env` dosyası ve `dotenv` kütüphanesi kullanarak Node.js ortamında ortam değişkenlerini yönetmeyi gösteren basit bir yapılandırma sistemidir.

Gizli bilgiler, API anahtarları veya proje ayarları `.env` dosyasında saklanır ve uygulama tarafından dinamik olarak kullanılır.

---

## Nasıl Kurulur?

1. Projeyi indir veya klonla.
2. Terminalde proje klasörüne geç:

   ``bash
   cd env-config-app
npm install dotenv
node index.js
