
## Proje Açıklaması
Bu proje, kullanıcıların seçtikleri bir dosyanın boyutunu hesaplayan bir uygulama sağlar. Kullanıcı dosya yüklediğinde, uygulama dosyanın boyutunu hesaplayacak ve kullanıcıya gösterecektir.

## Ekip/Kaynaklar
- **Geliştirici**: [ Quenn Exe]
- **Teknolojiler**:
  - Node.js
  - Express.js
  - Multer

## Proje Hedefleri
- Kullanıcının seçtiği dosyanın boyutunu hesaplamak.
- Kullanıcıya dosya yükleme imkanı sağlamak.
- Dosyanın boyutunu megabyte cinsinden göstermek.
