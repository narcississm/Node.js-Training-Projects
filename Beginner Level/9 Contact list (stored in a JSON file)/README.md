# Kişi Listesi (JSON Dosyasında Saklanan)

Bu proje, bir kişiyi ekleyebileceğiniz, güncelleyebileceğiniz, silebileceğiniz ve listeleyebileceğiniz bir API sunmaktadır. Kişi bilgileri JSON dosyasına kaydedilir.

## Kullanılan Teknolojiler
- Node.js
- express: Web sunucusu için
- fs: Dosya işlemleri için
