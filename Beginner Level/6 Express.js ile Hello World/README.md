# Express.js ile "Hello World" Projesi

Bu proje, Node.js ve Express.js kullanarak basit bir "Hello World" web uygulaması geliştirmektedir. Proje, bir web sunucusu başlatır ve tarayıcıda "Hello World" mesajını gösterir.

## Gereksinimler

- Node.js
- Express.js

## Kurulum

1. Projeyi bilgisayarınıza klonlayın veya yeni bir klasör oluşturun.
2. Terminal veya komut istemcisine şu komutları girin:



# NPM ile proje başlatın (package.json oluşturur)
npm init -y

# Express.js'i yükleyin
npm install express 


# Hedefler
- Express.js kullanarak temel bir web sunucusu oluşturmak.

- Bir GET isteği ile tarayıcıya basit bir mesaj göndermek.
